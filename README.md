# JavaTraining-Week4

## 概要
Spring Boot を用いて、簡単な REST API（GET / POST）を実装した課題の提出物である。  
Java および Spring Boot は学習を始めたばかりの状態で取り組んだ。

---

## 実行方法
```powershell
.\gradlew.bat bootRun
```
## 動作確認

動作確認の証跡として、screenshots/ フォルダにサーバ起動時のログおよび API 疎通結果のスクリーンショットを格納している。


## 提出にあたって苦労した点

Git や GitHub の操作に不慣れで、提出方法の理解に時間を要した。
特に、Pull Request の差分が提出物として扱われる点や、ローカルと GitHub 上の状態の違いを理解するまでに試行錯誤が必要であった。

## まとめ

本課題を通して、API 実装の基礎に加え、GitHub を用いた課題提出の流れを理解することができた。
